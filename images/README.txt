【ヒーロー背景画像の差し替え方法】

このフォルダに、以下の3つのファイル名で画像を置くと、
トップページの背景が自動で写真に切り替わります（数秒ごとにフェード切替）。

  hero-1.jpg … 1枚目（例：日本海・香住の海岸／朝陽）
  hero-2.jpg … 2枚目（例：地域の食・市場・港町の活気）
  hero-3.jpg … 3枚目（例：未来へ続く道・挑戦のイメージ）

・推奨サイズ：横 1920px 以上（横長 16:9 程度）
・形式：jpg（pngでも可。その場合は index.html 内の "hero-1.jpg" などの拡張子を変更）
・画像を置かない場合は、上質な濃紺グラデーションの背景として表示されます。

※ photo-AC等で選んだ画像を、上記のファイル名にリネームして置くだけでOKです。
